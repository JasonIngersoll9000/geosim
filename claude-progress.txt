### Architecture decisions
- Using bun (not npm/npx) for all package management — npm/npx are Windows binaries in WSL2
- Vitest configured with node environment (not jsdom) for server-side logic tests
- docs/ excluded from TypeScript compilation (reference docs, not compiled code)
- ESLint argsIgnorePattern: ^_ must be in .eslintrc.json (Vercel build requires it)
- prefer-const applies even when mutating object properties via bracket notation
- Concurrent TurnPlan validation must check every concurrent pair (i<j loop), not just vs primary
- JSX comment text: use {' // text '} not // text (react/jsx-no-comment-textnodes ESLint error)
- Inline style objects: put border shorthand BEFORE borderBottom, not after (shorthand overwrites specific)
- Frontend validation workflow: geosim-playwright skill → geosim-uiux-validation skill → user approves all 4 CATs → commit
